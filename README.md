# 📊 JobInsight Agent: 智能招聘市场分析与洞察系统

## 💡 项目简介 (Project Overview)
本项目是一个结合了 **RAG (检索增强生成)** 与 **Multi-Agent (多智能体)** 架构的轻量级数据分析平台。通过对真实岗位 JD 的语义解析与结构化提取，为求职者和业务决策者提供可视化的市场洞察大屏。

**核心解决痛点**：传统招聘数据分析依赖人工打标，且无法深度理解非结构化文本背后的技能图谱。本项目利用大语言模型实现了从“自然语言理解”到“业务指标可视化”的端到端自动化。

## ✨ 核心功能 (Key Features)
* 🤖 **多智能体协作流**: 设计了数据清洗、技术分析、薪资统计等协同 Agent 链路。
* 📚 **RAG 增强检索**: 本地挂载静态岗位知识库，精准召回核心技能要求。
* 📈 **自动化数据洞察**: 将非结构化的 JD 文本转化为结构化特征，并利用 Streamlit 生成动态图表。

## 🛠️ 技术栈 (Tech Stack)
* **大模型与框架**: Qwen, LangChain
* **数据分析与可视化**: Pandas, Streamlit
* **核心环境**: Python 3.10+
